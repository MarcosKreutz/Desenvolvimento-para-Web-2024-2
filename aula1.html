Exercícios Aula 1

O que é Hypertext Markup Language(HTML)?

HTML, ou Hypertext Markup Language, é a linguagem de marcação padrão usada para criar e estruturar páginas na web. Ela define a estrutura de um documento web através de uma série de elementos e tags que organizam o conteúdo, como texto, imagens, links e outros elementos multimídia.

Como e porquê surgiu o HTML?

O HTML surgiu na década de 1990 como uma solução para uma necessidade crescente de organização e compartilhamento de informações na web.

O que é eXtensible Hypertext Markup Language(XHTML)

XHTML, ou eXtensible Hypertext Markup Language, é uma versão mais rigorosa e bem definida do HTML, que combina as características do HTML com as regras e a estrutura do XML.

Qual a diferença de XHTML para HTML

A principal diferença entre XHTML e HTML está na forma como as regras de sintaxe e estrutura são aplicadas. Enquanto ambos são usados para criar e estruturar páginas web, XHTML é baseado em XML e adota uma abordagem mais rigorosa.

O que é eXtensible Markup Language

XML, ou eXtensible Markup Language, é uma linguagem de marcação desenvolvida para armazenar e transportar dados de forma estruturada e legível tanto por humanos quanto por máquinas. Foi criado pelo W3C (World Wide Web Consortium) e publicado em 1998.

Qual a diferença de HTML para XML

Em resumo, HTML é focado na apresentação e estrutura de documentos web para visualização em navegadores, enquanto XML é usado para armazenar e transportar dados de forma estruturada e legível, permitindo maior flexibilidade na definição de tags e validação rigorosa.
